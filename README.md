# Emoji Quiz

Десктопная викторина: угадай фильм, сериал или бренд по эмодзи.
Сделано на C# + WinForms + SQLite (EF Core).

## Что умеет приложение

- **Режим игры** — случайный вопрос (эмодзи), 4 варианта ответа, счёт, таймер на ответ.
- **Фильтр по категориям** — можно играть только по фильмам, сериалам или брендам.
- **Режим администратора** — добавление своих вопросов (эмодзи, ответ, категория) прямо в базу, без перезапуска приложения.
- **База данных SQLite** — все вопросы хранятся в файле `quiz.db`, создаётся автоматически при первом запуске и заполняется из `movies_ru_emoji.tsv`. Записи администратора сохраняются между запусками.
- **Проверка ввода** — пустые поля и дубликаты в админке не добавляются.

## Как я готовила данные

Для игры нужны русские названия фильмов и их эмодзи. 
Готового датасета с таким сочетанием не нашлось, 
поэтому я взяла два источника и соединила их.

С Kaggle взяла 253 фильма с эмодзи — но там только английские названия.
Русские названия нашла через IMDb: сначала по английскому названию 
искала код фильма, потом по коду брала русский перевод.

## Что получилось

Из 253 фильмов удалось получить 172 записи с русскими названиями. Остальные фильмы либо не удалось сопоставить с IMDb, либо для них отсутствовало русское название в базе IMDb.

Интересно: после нормализации названий (приведение к нижнему регистру и удаление лишних пробелов) находится на 3 фильма больше, чем при точном сравнении (было 169, стало 172).
Это потому что в одном датасете могло быть "the lion king", 
а в другом "The Lion King" — по сути одно и то же.

## Как запустить

Нужен .NET 10 SDK — скачать можно с сайта Microsoft.

Дальше:
1. Скачать репозиторий или склонировать:
   git clone https://github.com/gonkionline005-cloud/emoji-quiz.git
2. Зайти в папку `EmojiQuiz/EmojiQuiz`
3. Запустить через терминал: `dotnet run`
   (или открыть `.sln` в Rider / Visual Studio и нажать зелёную кнопку)

При первом запуске база данных создастся сама и заполнится из файла
movies_ru_emoji.tsv — ничего дополнительно делать не нужно.

## Структура репозитория

```
data-prep/                скрипт подготовки данных (Python)
  prepare_data.py
  movies_emojis.csv       исходный датасет с Kaggle
  movies_ru_emoji.tsv     результат подготовки данных

EmojiQuiz/EmojiQuiz/      исходники приложения (C#)
  MainForm.cs             главное меню
  GameForm.cs             логика игры
  AdminForm.cs            режим администратора
  ResultForm.cs           экран результата
  Db.cs                   работа с базой данных
  QuizContext.cs          настройка EF Core
  Question.cs             модель вопроса
  movies_ru_emoji.tsv     данные для заполнения базы при первом запуске
```
