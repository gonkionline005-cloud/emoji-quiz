# Emoji Quiz

Приложение-викторина: угадай фильм по эмодзи. Сделано на C# + WinForms + SQLite.

## Как я готовила данные

Для игры нужны русские названия фильмов и их эмодзи. 
Готового датасета с таким сочетанием не нашлось, 
поэтому я взяла два источника и соединила их.

С Kaggle взяла 253 фильма с эмодзи — но там только английские названия.
Русские названия нашла через IMDb: сначала по английскому названию 
искала код фильма, потом по коду брала русский перевод.

## Что получилось

Из 253 фильмов удалось получить 172 записи с русскими названиями. Остальные фильмы либо не удалось сопоставить с IMDb, либо для них отсутствовало русское название в базе IMDb.

Интересно: после нормализации названий (приведение к нижнему регистру и удаление лишних пробелов) находится на 3 фильма больше, чем при точном сравнении (было 169, стало 172).
Это потому что в одном датасете могло быть "the lion king", 
а в другом "The Lion King" — по сути одно и то же.

## Как запустить

Нужен .NET 10 SDK — скачать можно с сайта Microsoft.

Дальше:
1. Скачать репозиторий или склонировать:
   git clone https://github.com/gonkionline005-cloud/emoji-quiz.git
2. Зайти в папку EmojiQuiz
3. Запустить через терминал: dotnet run
   (или открыть в Rider и нажать зелёную кнопку)

При первом запуске база данных создастся сама и заполнится из файла
movies_ru_emoji.tsv — ничего дополнительно делать не нужно.
